# 3D-Interaction

Nom de l'équipe : JungleDiff

Florent Bélot - Shanti Noel - Alexis Dubarry - Allah-Eddine Cherigui

On a un projet avec un simulateur, un play mode et le build qui fonctionne mais on ne peut pas intéragir avec le cube.

Pour l'instant notre idée pour le projet serait un jeu où on joue un aventurier dans un temple qui aura pour but de trouver un trésor dans les entrailles du temple. Mais il devra réussir à passer des épreuves et à éviter des pièges (un peu comme Indiana Jones).
